## 🎯 Key Innovations

### **1. Capability-Based Discovery**
```rust
// Not this (hardcoded):
let songbird = start("songbird", 8080);

// But this (capability-based):
let comms = registry.find_by_capability("discovery")?;
```

**Benefits**:
- Future-proof
- Flexible
- Testable
- Evolvable

---

### **2. Graph-Based Orchestration**
```toml
# Declarative TOML definitions
[graph]
name = "deploy-tower"

[[nodes]]
id = "start_songbird"
primal.by_capability = "discovery"
operation.name = "start"
```

**Benefits**:
- Declarative
- Easy to modify
- Clear dependencies
- Powerful abstraction

---

### **3. Runtime Primal Discovery**
- Scans `/tmp/*.sock` for primal sockets
- Queries capabilities via JSON-RPC
- No compile-time coupling
- Zero configuration required
