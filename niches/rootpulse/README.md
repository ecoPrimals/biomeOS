## Overview

RootPulse is NOT a new primal—it's an **emergent capability** that arises from coordinating existing primals for **universal temporal tracking**. It demonstrates BiomeOS's power to create complex systems through simple primal composition.

### What RootPulse Does

Traditional version control (Git) is monolithic. RootPulse **emerges** from primal coordination:

- **Version Control**: Code commits (traditional VCS)
- **Universal Time**: Art creation, life events, performances
- **Temporal Tracking**: Experiments, game states, particle collisions
- **DAG Present/Future**: rhizoCrypt (branching possibilities, 10-100x faster)
- **Linear Past**: LoamSpine (immutable history, provable)
