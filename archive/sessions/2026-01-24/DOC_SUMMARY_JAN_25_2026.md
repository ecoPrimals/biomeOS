## 📝 Documentation Standards

### File Naming
- `STATUS_<DATE>.md` - Status snapshots
- `<TOPIC>_<DATE>.md` - Time-stamped docs
- `<TOPIC>.md` - Living docs (no date)

### Headers
```markdown
# Title
**Last Updated**: January 25, 2026
**Status**: 🚀 Active / ✅ Complete / 📋 Planned
```

### Sections
- Always include: Overview, Status, Next Steps
- Use emojis for visual scanning
- Link to related docs
- Keep TOC for docs > 100 lines
