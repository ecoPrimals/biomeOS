### CLI Framework

**Implementation**: Clap v4 with derive macros

**Structure**:
```rust
#[derive(Parser)]
#[command(name = "biomeos")]
#[command(about = "🧠 biomeOS Universal Nucleus & Orchestrator")]
#[command(version = env!("CARGO_PKG_VERSION"))]
struct Cli {
    #[arg(short, long, global = true)]
    verbose: bool,

    #[arg(long, global = true, default_value = "info")]
    log_level: String,

    #[command(subcommand)]
    mode: Mode,
}
```

**Global Options**:
- ✅ `-v, --verbose`: Enable verbose logging
- ✅ `--log-level`: Set log level (trace, debug, info, warn, error)
- ✅ Applies to all modes

**Status**: ✅ PROFESSIONAL - Modern Rust CLI patterns

## 🚀 USAGE EXAMPLES

### Start Neural API Server
```bash
biomeos neural-api \
  --graphs-dir ./graphs \
  --family-id nat0 \
  --socket /tmp/biomeos-neural-api.sock
```

### Execute Deployment
```bash
# Validate only
biomeos deploy graphs/production.toml --validate-only

# Dry run
biomeos deploy graphs/production.toml --dry-run

# Execute
biomeos deploy graphs/production.toml
```

### Start API Server (Unix Socket)
```bash
biomeos api --socket /tmp/biomeos-api.sock --unix-only
```

### Run Health Diagnostics
```bash
# Quick check
biomeos doctor

# Detailed text output
biomeos doctor --detailed

# JSON output for automation
biomeos doctor --format json

# Check specific subsystem
biomeos doctor --subsystem network
```

### Verify Genetic Lineage
```bash
biomeos verify-lineage /path/to/spore --detailed
```
