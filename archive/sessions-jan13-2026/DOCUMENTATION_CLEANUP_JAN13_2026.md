## 📝 Maintenance Guidelines

### When to Archive

- **Old session docs** - After new session complete
- **Superseded docs** - When newer version exists
- **Milestone docs** - After milestone achieved
- **Duplicate docs** - Keep latest version only

### When to Keep at Root

- **Essential docs** - README, START_HERE, STATUS
- **Latest session** - Most recent work
- **Active plans** - Current strategies
- **Architecture** - Core system design
- **Integration** - Active integration work
